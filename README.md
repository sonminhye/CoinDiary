# CoinDiary
* 비트코인 투자에 대한 다이어리 작성 프로그램

### 사용기술스택

* Spring Boot
* JPA-Hibernate
* H2Database
* Upbit Open Api 연동
* Thymeleaf

### 구현내용
* 코인에 대한 평단가/매수금 등 정보 등록 및 수정
* 마켓에 있는 모든 코인정보 불러오기 기능(업비트 Api 연동)

### 추가적으로 구현해보고 싶은 부분
* 본인 계정과 연동하여 내 거래내역 불러와, 해당 거래내역으로 간단하게 다이어리 등록
* 로그인/로그아웃 기능(회원정보관련 기능) - SpringSecurity
* ...
